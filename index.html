        🧠 설계 모드 셋업 (1회, 약 3분)

        PC 가 꺼져 있어도 폰의 Claude 가 내 작업 지식 폴더를 읽고 함께 설계하게 만듭니다. ①부터 ⑥까지 위에서부터 순서대로 따라 하면 됩니다.

        ① 프로젝트 만들기

        1. 아래 버튼 → 브라우저로 claude.ai 가 열림 (Claude 계정 로그인)
        2. 새 프로젝트 버튼 → 이름 "🧠 설계 모드" 입력 → 만들기
        ⚠️ 핵심은 같은 Claude 계정 — 만들 때와 열 때 계정이 다르면 "액세스 권한 없음"이 뜹니다. 폰이든 PC 든 같은 계정이면 OK. 잘 모르겠으면 이 버튼으로 열린 브라우저에서 만드세요.

        ② Drive 연결 — 프로젝트 안 "파일" 영역에서 (= 지식. 화면엔 메모리/지침/파일로 보임)

        1. 프로젝트 "🧠 설계 모드" 를 열기 (폰이 불편하면 PC 브라우저, 같은 계정)
        2. 파일 영역의 ＋ 누르기
        3. Drive 선택 — 처음이면 Google 로그인 + 권한 모두 허용
        4. 검색창에 Claude_GMD — 폴더가 뜨면 폴더 추가. 폴더 선택이 안 되는 화면이면 DESIGN_MANIFEST 검색해 그 파일 추가
        • ＋에 Drive 선택지가 없으면 = 계정에 Drive 커넥터 미연결. PC 웹에서: 채팅 입력칸 옆 ＋ → 커넥터 → 커넥터 관리 (또는 왼쪽 사용자 지정 → 커넥터) → ＋ → Google Drive → 연결 → 권한 허용 → 프로젝트로 돌아와 파일 ＋ 다시 (폰 커넥터 설치는 베타 — PC 웹에서 한 번 연결하면 폰에도 적용)
        • Drive 추가는 비공개 프로젝트에서만 가능
        • ✅ 파일 목록에 폴더(또는 매니페스트 파일)가 떠 있으면 성공

        ③ 지시문 붙여넣기

        1. 아래 버튼 → 지시문이 복사됨
        2. 프로젝트 화면의 지침(Instructions) 설정 칸 열기 → 붙여넣기 → 저장
        이 글이 "내 지식 폴더를 읽고 설계만 해라"라고 Claude 에게 시키는 내용입니다.

        ④ 프로젝트 주소 저장

        1. 브라우저 주소창의 프로젝트 주소(https://claude.ai/project/…)를 길게 눌러 복사
        2. 아래 버튼 탭 → 붙여넣고 확인
        이때부터 설계 탭의 🧠 버튼이 이 프로젝트로 바로 연결됩니다.

        ⑤ 연결 확인 (중요 — 꼭 한 번)

        1. 아래 첫 버튼 → 확인 질문이 복사됨
        2. 둘째 버튼으로 설계 대화 열기 → 질문 붙여넣고 전송
        ✅ 매니페스트 요약이 나오면 셋업 완료 / ❌ "못 읽는다"고 답하면 아래 ⑥.

        ⑥ 안 될 때 (문제 해결)

        • 파일 ＋에 Drive 가 끝내 안 떠도 OK — 대화 안에서 입력칸 옆 ＋ → 커넥터 → Google Drive 켜기 → "Google Drive 에서 DESIGN_MANIFEST.md 검색해 요약해줘" 전송. 검색으로 읽히면 파일 추가 없이도 정상 작동 (실측 검증된 경로)
        • Claude 가 "도구/커넥터가 없다"고 함 → 대화에 뜨는 "커넥터 검색 허용" 물음에 허용
        • 폴더는 연결됐는데 파일을 못 읽음 → ② 에서 Claude_GMD/ClaudeConfig 폴더로 다시 연결 (하위 폴더가 안 읽히는 환경 — 거기에도 같은 안내 파일이 있음)

        평소 사용: 설계 대화에서 스펙 확정 → 복사 → 설계 탭 📋 스펙 붙여넣기 → PC 켜지면 자동 전송 → 대시보드 ▶ 진행.
